Cara Penggunaan

  1. Siapkan file Excel dengan daftar pemeriksaan (format seperti contoh)
  2. Unggah file menggunakan form di bawah
  3. Sistem akan secara otomatis mengelompokkan pemeriksaan berdasarkan jenisnya
  4. Download hasil yang sudah dikelompokkan dalam format Excel
Unggah File Excel
Seret file Excel ke sini atau klik untuk memilih

Format file yang didukung: .xlsx, .xls
